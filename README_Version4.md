# caligrafia8

Aplicación educativa basada en el método **"De Letra Fea a Escritura Elegante"**, diseñada para ayudar a mejorar la caligrafía en 30 días mediante módulos interactivos, ejercicios guiados y seguimiento del progreso.

Tecnologías principales:
- React Native (Expo) para aplicaciones móviles iOS y Android
- Next.js para la versión web PWA
- Firebase (Auth, Firestore, Storage, Cloud Functions)
- Monorepo con pnpm y Turborepo

## Setup rápido

Requisitos:
- Node.js 18+
- pnpm
- Firebase CLI
- Expo CLI
